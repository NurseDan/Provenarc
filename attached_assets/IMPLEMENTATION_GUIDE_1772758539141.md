## Deployment Instructions

### Step 1: File Integration

Upload the four HTML files to your website:
```
/marine/why-us.html → marine-why-us.html
/marine/process.html → marine-process.html
/auto/why-us.html → auto-why-us.html
/auto/process.html → auto-process.html
```

### Step 2: Update Navigation Links

Update your existing Marine and Auto division pages to link to the new pages instead of the Aero pages.

**Find and replace in Marine division navigation:**
- Old: /aero/why-us → New: /marine/why-us
- Old: /aero/process → New: /marine/process

**Find and replace in Auto division navigation:**
- Old: /aero/why-us → New: /auto/why-us
- Old: /aero/process → New: /auto/process

### Step 3: Update Internal Links

Each division page includes a CTA button linking to "/contact". Ensure this path is correct for your website structure, or update to the appropriate contact page URL.

### Step 4: SEO Optimization

Each page includes:
- Title tags optimized for division-specific keywords
- Meta descriptions tailored to each industry
- Header hierarchy (H1, H2, H3) for SEO structure
- Mobile-responsive design

Consider adding:
- Division-specific imagery (vessels, collector cars)
- Customer testimonials from each industry
- Case studies specific to marine/auto sectors

### Step 5: Conversion Tracking

Add tracking codes to CTA buttons:
- "Get Started" buttons
- "Schedule Consultation" buttons
- Contact form submissions

Track conversion rates by division to optimize marketing spend.
